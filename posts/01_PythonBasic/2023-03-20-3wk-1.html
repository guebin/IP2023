a+[4]와 a.append(4)의 차이점은?

(관찰1)

a=[1,2,3]

[1, 2, 3]

(관찰2)

a=[1,2,3]

a ## a자체가 변화함 
[1, 2, 3, 4]

비슷해보이지만 굉장히 미묘한 차이가 있음

a.append(4): a에 4를 append하라 \(\to\) a가 변함

a+[4]: a와 4를 연산하라

리스트 컴프리헨션

- 예제: \(2^0, 2^1, 2^2, 2^3\)를 원소로 가지는 리스트를 생성하라.

(풀이1) 직접입력

x= [2**0, 2**1, 2**2, 2**3] 
x
[1, 2, 4, 8]

(풀이2) for문을 이용함

x=[] 
for i in [0,1,2,3]:
    x.append(2**i) 
x
[1, 2, 4, 8]

(풀이3) for문을 이용함

x=[] 
for i in [0,1,2,3]:
    x = x+[2**i]
x
[1, 2, 4, 8]

(풀이4) for문을 이용함

x=[] 
for i in [0,1,2,3]:
    x += [2**i]
x
[1, 2, 4, 8]

(풀이5) 리스트컴프리헨션을 이용한 풀이

x= [2**i for i in [0,1,2,3]] 
x
[1, 2, 4, 8]

- 리스트컴프리헨션의 문법 암기방법 - 집합에서 조건제시법을 연상 - \(\{2^0,2^1,2^2,2^3\}=\{2^i: i \in \{0,1,2,3\} \}\)

- 리스트컴프리헨션이란? - 리스트를 매우 효율적으로 만드는 테크닉 - for문에 비하여 가지고 있는 장점: (1) 코드가 간결하다 (2) 빠르다
